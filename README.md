# 진메현 시리즈

진메현 시리즈 웹 리더입니다. Next.js(TypeScript) 정적 사이트로 편·장 단위 열람과 페이지 넘김을 지원합니다.

## 로컬 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

정적 결과물은 `out/`에 생성됩니다.

## GitHub Pages

`main` 브랜치 push 시 Actions로 자동 배포됩니다. 저장소 Settings → Pages → Source를 **GitHub Actions**로 설정하세요.
